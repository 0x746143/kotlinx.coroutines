fun Dispatchers.resetMain()
Content copied to clipboard

Resets state of the Dispatchers.Main to the original main dispatcher. For example, in Android Main thread dispatcher will be set as Dispatchers.Main. Used to clean up all possible dependencies, should be used in tear down (@After) methods.

It is unsafe to call this method if alive coroutines launched in Dispatchers.Main exist.
