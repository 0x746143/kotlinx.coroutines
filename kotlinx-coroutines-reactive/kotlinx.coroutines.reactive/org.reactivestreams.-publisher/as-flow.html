asFlow

@FlowPreview @JvmOverloads @JvmName("from") fun <T : Any> Publisher<T>.asFlow(
    batchSize: Int = 1
): Flow<T>

Transforms the given reactive Publisher into Flow. Backpressure is controlled by batchSize parameter that controls the size of in-flight elements and Subscription.request size.

If any of the resulting flow transformations fails, subscription is immediately cancelled and all in-flights elements are discarded.
