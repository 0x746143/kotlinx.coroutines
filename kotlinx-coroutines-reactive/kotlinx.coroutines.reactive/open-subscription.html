fun <T> Publisher<T>.openSubscription(request: Int = 1): ReceiveChannel<T>
Content copied to clipboard

Subscribes to this Publisher and returns a channel to receive the elements emitted by it. The resulting channel needs to be cancelled in order to unsubscribe from this publisher.

Parameters

request

how many items to request from the publisher in advance (optional, a single element by default).

This method is deprecated in the favor of Flow. Instead of iterating over the resulting channel please use collect:

asFlow().collect { value ->
    // process value
}
