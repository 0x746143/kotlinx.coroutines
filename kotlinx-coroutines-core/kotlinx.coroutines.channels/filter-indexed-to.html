filterIndexedTo

inline suspend fun <E, C : MutableCollection<in E>> ReceiveChannel<E>.filterIndexedTo(
    destination: C,
    predicate: (index: Int, E) -> Boolean

Deprecated: Channel operators are deprecated in favour of Flow and will be removed in 1.4.x

Appends all elements matching the given predicate to the given destination.

Parameters

predicate -

function that takes the index of an element and the element itself and returns the result of predicate evaluation on the element.

The operation is terminal. This function consumes all elements of the original ReceiveChannel.
