next

abstract operator fun next(): E (source)

Retrieves the element from the current iterator previously removed from the channel by preceding call to hasNext or throws IllegalStateException if hasNext was not invoked. next should only be used in pair with hasNext:

while (iterator.hasNext()) {
    val element = iterator.next()
    // ... handle element ...
}

This method throws ClosedReceiveChannelException if the channel isClosedForReceive without cause. It throws the original close cause exception if the channel has failed.
