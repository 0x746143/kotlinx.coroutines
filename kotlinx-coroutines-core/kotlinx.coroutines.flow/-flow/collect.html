collect

abstract suspend fun collect(
    collector: FlowCollector<in T>
): Unit (source)

Accepts the given collector and emits values into it.

A valid implementation of this method has the following constraints:

  1. It should not change the coroutine context (e.g. with withContext(Dispatchers.IO)) when emitting values. The emission should happen in the context of the collect call.

Only coroutine builders that inherit the context are allowed, for example the following code is fine:

coroutineScope { // Context is inherited
    launch { // Dispatcher is not overridden, fine as well
        collector.emit(someValue)
    }
}
  1. It should serialize calls to emit as FlowCollector implementations are not thread safe by default.
