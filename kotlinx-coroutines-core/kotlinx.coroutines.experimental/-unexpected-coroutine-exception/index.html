UnexpectedCoroutineException

class UnexpectedCoroutineException : IllegalStateException (source)

Represents an exception in the coroutine that was not caught by it and was not expected to be thrown. This happens when coroutine is cancelled, but it completes with the different exception than its cancellation cause was.
