onCancellation

protected open fun onCancellation(cause: Throwable?): Unit (source)

This function is invoked once when this coroutine is cancelled or is completed, similarly to invokeOnCompletion with onCancelling set to true.

The meaning of cause parameter:

  • Cause is null when job has completed normally.
  • Cause is an instance of CancellationException when job was cancelled normally. It should not be treated as an error. In particular, it should not be reported to error logs.
  • Otherwise, the job had failed.
