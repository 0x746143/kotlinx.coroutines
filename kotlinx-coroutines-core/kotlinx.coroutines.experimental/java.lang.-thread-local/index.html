asContextElement
fun <T> ThreadLocal<T>.asContextElement(
    value: T = get()
): ThreadContextElement<T>

Wraps ThreadLocal into ThreadContextElement. The resulting ThreadContextElement maintains the given value of the given ThreadLocal for coroutine regardless of the actual thread its is resumed on. By default ThreadLocal.get is used as a value for the thread-local variable, but it can be overridden with value parameter.
