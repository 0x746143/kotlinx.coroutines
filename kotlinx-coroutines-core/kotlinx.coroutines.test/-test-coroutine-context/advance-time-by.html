advanceTimeBy

fun advanceTimeBy(
    delayTime: Long,
    unit: TimeUnit = TimeUnit.MILLISECONDS
): Long (source)

Platform and version requirements: JVM

Moves the CoroutineContext’s virtual clock forward by a specified amount of time.

The returned delay-time can be larger than the specified delay-time if the code under test contains blocking Coroutines.

Parameters

delayTime - The amount of time to move the CoroutineContext’s clock forward.

unit - The TimeUnit in which delayTime and the return value is expressed.

Return The amount of delay-time that this CoroutinesContext’s clock has been forwarded.
