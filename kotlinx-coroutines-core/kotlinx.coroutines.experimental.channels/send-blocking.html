fun <E> <ERROR CLASS><E>.sendBlocking(element: E): Unit (source)

Platform and version requirements: JVM

Adds element into to this channel, blocking the caller while this channel Channel.isFull, or throws exception if the channel Channel.isClosedForSend (see Channel.close for details).

This is a way to call Channel.send method inside a blocking code using runBlocking, so this function should not be used from coroutine.
