filterIndexed

fun <E> ReceiveChannel<E>.filterIndexed(
    context: CoroutineContext = Dispatchers.Unconfined,
    predicate: suspend (index: Int, E) -> Boolean
): ReceiveChannel<E>

Returns a channel containing only elements matching the given predicate.

Parameters

predicate -

function that takes the index of an element and the element itself and returns the result of predicate evaluation on the element.

The operation is intermediate and stateless. This function consumes all elements of the original ReceiveChannel.
