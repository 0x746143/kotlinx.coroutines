createCopy
fun createCopy(): TimeoutCancellationException?

Creates a copy of the current instance. For better debuggability, it is recommended to use original exception as cause of the resulting one. Stacktrace of copied exception will be overwritten by stacktrace recovery machinery by Throwable.setStackTrace call. An exception can opt-out of copying by returning null from this function.
